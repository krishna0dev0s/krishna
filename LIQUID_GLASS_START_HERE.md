## üîß Simple Customization

### Change Effect Type
```jsx
<LiquidGlassCard effectType="neon">  {/* Change to: glow, gradient, liquid, frost */}
  Content
</LiquidGlassCard>
```

### Change Intensity
```jsx
<LiquidGlassCard glowIntensity="lg">  {/* xs, sm, md, lg, xl */}
  Content
</LiquidGlassCard>
```

### Add Tailwind Styles
```jsx
<LiquidGlassCard className="p-8 rounded-2xl shadow-2xl">
  Content
</LiquidGlassCard>
```

### Enable Advanced Features
```jsx
<LiquidGlassCard
  liquidEffect={true}      // Morphing blobs
  borderGradient={true}   // Cursor tracking
  draggable={true}        // Drag enabled
  animate={true}          // Animations on
>
  Advanced card
</LiquidGlassCard>
```
